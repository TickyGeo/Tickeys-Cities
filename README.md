Tickeys' Cities

Preview: ![preview](preview.png)

In this game you need to guess the cities on the photoes. There are letters at the bottom on which you need to click. First levels are easy, but they become harder and harder. In the last levels you'll need to answer some questions to unlock the letters and guess where they are hidden.

I used HTML, internal JS and CSS for this game. I created many HTML pages in which I wrote my JS code.
